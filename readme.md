Install the compiler
```
sudo apt-get install ghc
```

Compile and run
```
$ runghc hello.hs 
Hello, World!
```

## Examples
```haskell
Prelude> succ 5
6
Prelude> truncate 6.59
6
Prelude> round 6.59
7
Prelude> sqrt 2
1.4142135623730951
Prelude> not (5 < 3)
True
Prelude> gcd 21 14
7
```
